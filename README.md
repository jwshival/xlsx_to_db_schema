### 3. XLSX to DB Schema APIs:

`
./console {:adapter} {:sourcePATH} {:destPATH}
`

- adapter: mysql or postgresql
- sourcePATH: Storage location of csv or xslx files.
- destPATH: Storage location of output db schema.
